## CSS基础



### 定义

1. 层叠样式表（Cascading Style Sheet，简称：CSS）是为网页添加样式的代码。
2. CSS不是一种标记语言，是一门样式表语言。



### CSS规则集

1. 选择器（Selector），位于规则集开始，选择一个或多个需要添加样式的元素或类或id或属性等：
   ```css
   p {}
   .class {}
   #id {}
   img[src] {} // 选择的是`<img src="images/logo.png" >`而不是`<img>`
   ```

2. 声明（Declaration），用来指定添加样式元素的属性：
   ```css
   {
    color: red;
   }
   ```

3. 属性（Properties），改变元素样式的途径，比如：`color`，`font-size`等

4. 属性值（Property value），在属性的右边，冒号后面即属性的值。

5. 每个规则集（除了选择器的部分）都应该包含在成对的大括号里（{}）。
   
6. 在每个声明里要用冒号（:）将属性与属性值分隔开。

7. 在每个规则集里要用分号（;）将各个声明分隔开。



### 一切皆盒子

1. CSS编写即围绕一个个盒子展开的————设置尺寸、颜色、位置等等，页面里大部分HTML元素都可以被看作若干层叠的盒子。
2. CSS布局主要是基于盒模型的。每个占据页面空间的块都有如下属性：
   1. padding：即内边距，围绕着内容（比如段落）的空间。
   2. border：即边框，紧接着内边距的线。
   3. margin：即外边距，围绕元素外部的空间。

3. `<body>`元素是块级元素，意味着他占据了页面的空间并且能够赋予外边距和其他改变间距的值。
4. 而图片是内联元素，不具备块级元素的一些功能。所以为了使图形有外边距，则必须改变元素类型为块级元素：`display:block;`
